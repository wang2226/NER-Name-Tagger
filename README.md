# NER Name Tagger
Features:
* POS
* chunk
* token index
* START
* END
* previous token
* next token
* use "@@" to refer to previous tag if doesn't know what tags will be assigned
* stop words from NLTK
* is_geo_place from GeoText
* case
* last_char
* is_last_name 
* is_city

## To run
```sh
$ sh run.sh
```
